# Darksay
A simple command-line tool to display random dark quotes directly in your terminal.

### Requirements
- ruby

### Installation
```bash
git clone https://github.com/Blziko/darksay
chmod +x darksay/darksay.rb
sudo ln -sf $HOME/darksay/darksay.rb /usr/bin/darksay
```

### Usage
```bash
darksay
```
